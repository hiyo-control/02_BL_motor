# Chapter 2 IHM07M1 PWM output program
## How to build program
1. Open "terminal"
2. Enter "make clean"
3. Enter "make"

## How to execute exe-file
1. Open "terminal"
2. Enter "sudo ./build/main"
3. Enter your password if it is needed
